Проект аналитика
Я постаралась создать проект, который можно использовать на практике уже сейчас. Он подходит для работы экономистов, когда им нужно определить как температурные условия, климат влияют на экономику страны, на её население.
Файл rainfall.py показывает диаграмму с уровнем осадков.
Файл highs_Lows.py показывает диаграмму температур(max and min) по данным одной страны.
Файл highs_Lows2.py показывает диаграмму 2 стран. Пытаться внести на диаграмму 3 страну уже уменьшает понимание и эффективность полученной информации.
Файл global_gdp.py показывает на карте ВВП стран в Долларах(страны разделены на три группы по размеры ВВП).
Файл world_population.py показывает на карте население стран(страны разделены на группы по величине населения).
Файл countriess.py сортирует ключи стран(с которыми работает программа) в алфавитном порядке, так же ф этом файле происходит переход от 3-х буквенного кода из файла population_data.json  к 2-х буквенному коду, который использыет Pygal.
Файл country_tests.py тестируем работоспособность функции get_country_code() в файле country_codes.py
При просмотре файлов global_gdp.svg и т.д. лучше использовать Mozilla Firefox(т.к. будет более наглядно) и не ипользовать Internet Explorer(при наведении на карту не появляется информации).
